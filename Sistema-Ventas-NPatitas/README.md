# Sistema de Ventas - N' PATITAS

Sistema de gestión de ventas desarrollado en C++ para la farmacia veterinaria N' PATITAS.

## 📁 Estructura del Proyecto
Sistema-Ventas-NPatitas/
├── bin/ # Ejecutables compilados
├── data/ # Archivos de datos (productos.txt, ventas.txt)
├── docs/ # Documentación
└── src/ # Código fuente
├── main.cpp
├── Producto.h / Producto.cpp
├── Venta.h / Venta.cpp
├── Archivo.h / Archivo.cpp
└── Menu.h / Menu.cpp

## 🛠️ Compilación

```bash
# Compilar en Windows con MinGW
g++ -std=c++17 src/*.cpp -o bin/SistemaVentas.exe

# Compilar en Linux
g++ -std=c++17 src/*.cpp -o bin/SistemaVentas
# Windows
cd bin
SistemaVentas.exe

# Linux
cd bin
./SistemaVentas
